# Python 오토클릭 프로그램

이 프로젝트는 간단한 Python GUI 오토클릭 프로그램입니다.

## 기능

- F4: 오토클릭 시작
- F5: 오토클릭 정지
- 초당 클릭 횟수(CPS) 설정 및 저장
- 좌/우클릭 선택

## 설치 방법

1. Python 3.x가 설치되어 있어야 합니다.
2. 이 폴더에서 다음 명령어로 필요한 패키지를 설치합니다.

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
python autoclicker.py
```

창이 뜨면:

1. 원하는 **CPS**와 **마우스 버튼(좌/우클릭)** 을 선택합니다.
2. **설정 저장** 버튼을 눌러 저장합니다.
3. 마우스를 원하는 위치에 두고 **F4**를 누르면 오토클릭이 시작됩니다.
4. **F5**를 누르면 오토클릭이 정지됩니다.

동일 폴더의 `config.json` 파일에 설정이 자동으로 저장됩니다.
